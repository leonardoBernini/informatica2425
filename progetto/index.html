Descrizione del progetto

Questo semplice progetto permette di misurare la temperatura utilizzando il sensore LM35 e una scheda Arduino UNO. Il sensore invia un segnale analogico proporzionale alla temperatura, che viene letto e convertito in gradi Celsius da Arduino. Il risultato viene mostrato sul monitor seriale ogni secondo.

Componenti utilizzati

  • 1x Arduino UNO
  • 1x Sensore di temperatura LM35
  • Cavi jumper
  • Breadboard
  • USB cable

Codice Arduino

int lm35Pin = A0;
float temperatura;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int valore = analogRead(lm35Pin);
  temperatura = valore * 0.48828125;
  Serial.print("Temperatura: ");
  Serial.print(temperatura);
  Serial.println(" °C");
  delay(1000);
}
